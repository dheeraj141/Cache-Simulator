# Cache-Simulator
This is a cache simulator written in C which takes as input the memory trace file. 


commands to run the code 

1: make clean 

2: make sim 

3: <p> ./sim <cache_size> <associativity> <block_size> <eviction policy> <write policy> <trace file>  </p>
